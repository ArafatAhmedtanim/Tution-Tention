        Simple & most efficient solution -

        A number can be of power of 2 only if the bit wise AND of the given number and number -1 is 0

        e.g.
        Say given number is n = 8

        8421 - Bit representation of number
        1000 - 8
        0111 - 7 (8-1)
        -------
        0000 - Result is 0 so the given number is Power of 2

        Another example

        0111 - 7
        0110 - 6 (7-1)
        -------
        0110 - Result is not 0 so the given number is not Power of 2

        Pseudo code
        Say n is the given number
        if(n==0) then NO
        else
        m = n-1;
        res = n&m;
        if res==0 then YES
        else NO

        Note - Range is 0<=N<=10^18 so use the data type which can hold this range, I have used BigInteger in Java.

        Cheers
        Td

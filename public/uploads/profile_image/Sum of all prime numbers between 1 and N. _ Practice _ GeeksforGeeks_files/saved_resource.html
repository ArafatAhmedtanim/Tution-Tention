                                              RED_ACE Monisha Margaret 6 months ago

                                              The given range for N is 10^6 whereas you are constructing sieve for max range 10^5.
                                              So your code will give segmentation fault for accessing A[N-1] (i.e N=2,00,000) .
                                              Besides that you can optimise your code by storing the sum of primes from 2 to i in sum[i] so that you don't have to sum it up again and again by traversing 10^6 elements.
